## AIM:

## DESIGN STEPS:

### Step 1:
Create a new frame.

### Step 2:

### Step 3:

### Step 4:
Import images as needed.

### Step 5:
Create pages based on your need and link them.

### Step 6:

Validate the HTML and CSS code.

Publish the website in the given URL.
